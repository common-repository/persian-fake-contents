== Description ==
**محتوای جعلی فارسی**
A tool for persian wordpress developers to create Fake posts in Persian/Farsi language.
این یک ابزار ساده و رایگان برای طراحان سایت با سیستم مدیریت محتوای وردپرس است که این امکان را فراهم می‌آورد تا مطالب جعلی به زبان فارسی ایجاد کنند.



= Features =
* publish fake posts and pages and custom post types
* showing published fake posts
* delete all created fake posts


= Compatibility =
* آخرین نسخه وردپرس


== Frequently asked questions ==

= راهنمای نصب =
1. پلاگین را نصب کرده و فعال کنید
2. تعداد پست مورد نظرتان را وارد نمایید، و کلیک کنید، بلافاصله به همین تعداد پست جعلی با محتوای فارسی و عکس زمینه‌ی رندوم ایجاد و منتشر می‌شود
3. این مرحله را به تعداد مورد نیازتان می‌توانید تکرار کنید
4. تمامی پست‌های ایجاد شد از طریق همین پلاگین قابل حذف هستند
